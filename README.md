# Pricing-Premium-off-of-simulated-claim-and-Severity-data
This project's goal was to improve R studio skills by simulating a data set and doing an actuarial analysis of it. It includes simulating data sets using poisson,lognormal,and weibull distribution, fitting lognormal,gamma,and weibull distributions to data sets, calculating VaR and TVaR, and estimating total premium.
